<html> Tag

Root of the HTML document. It stores all of the HTML elements in exception of the <!DOCTYPE> tag.

<html lang="en">
<head>
</head>
<body>
</body>
</html>


<!DOCTYPE html> Tag

It tells the browser to use HTML5. Commonly seen at the first line of HTML code.

<!DOCTYPE html>
<html lang="en">
<head>
</head>
<body>
</body>
</html>



<head> Tag

Used as a container to hold the metadata of the HTML. It's usually written between the html and body tag.

<head>
<title>Document Title</title>
</head>


<body> Tag

Used to display the main contents of the HTML.

<body>
<h1>Sample Text</h1>
</body>


<H1> Tag

Used to display the title headings. It ranges from 1 to 6.

<h1>Sample Header</H1>//Biggest
<h6>Sample Header</H6>//Smallest


<p> Tag

Used to Display a paragraph of text.

<p>The quick brown fox jump over the lazy dog. The quick brown fox jump over the lazy dog. The quick brown fox jump over the lazy dog. The quick brown fox jump over the lazy dog.</p>


<br> Tag

Used for line break. Good for seperating texts without opening and closing tags over and over again. <br> are empty tags so there's no need for an end tag

<p>The quick brown fox jump over the lazy dog.<br> The quick brown fox jump over the lazy dog.<br> The quick brown fox jump over the lazy dog.<br> The quick brown fox jump over the lazy dog.</p>


<a> Tag

This tag creates a hyperlink and is commonly used with href attribute. Without href, it will only serve as a placeholder for hyperlink.

<a href="youtube.com"></a>


<img> Tag

Used to put an image in HTML followed by src to indicate the path of the image to use. If the image can't be displayed, alt attribute can be used to put a text in the image's place.

<img src="sans.png" alt="picture of sans from undertale.">


<style> Tag

Used to define a stylesheet or CSS that the HTML can use for design. It can be done internal or external of the HTML document.

<style>
body {
background-color: blue;
</style>


<div> Tag

It defines a division tag which seperates the tag from other tags. Can be easily stylized using CSS.

<div class="sampleDiv">
<h2>This is a heading of a sampleDiv.</h2>
<p>This is a text of sampleDiv</p>
</div>


<!----> Tag

Displays a comment inside the HTML file. CSS use /* */ for commenting

<div class="sampleDiv">
<h2>This is a heading of a sampleDiv.</h2>
<p>This is a text of sampleDiv</p>
</div>
<!--This is a comment-->


<li> Tag

Used to define a list. Often used inside ordered list <ol> or unordered list <ul> respectively.

<ol>
<li>Coffee</li>
<li>Tea</li>
</ol>
<ul>
<li>Coffee</li>
<li>Tea</li>
</ul>


<table> Tag

Displays a table in HTML. The <table> consists of <td> , <tr> , and <th>.

<table>
<tr>
<th>Month</th>
</tr>
<tr>
<td>January</td>
</tr>


<footer> Tag

Used to display a footer on the document. Mainly seen below a website.

<footer>
<p>Author: Intal
<a href="mailto:abc@example.com"></a></p>



CSS

border Property

It makes a border of a certain element.

p{
border: 1px solid green;
}


color Property

It is used to specify a color of the text.

h1{
color:red;
}



padding Property

It adjust the space between the content and the border of an element.

div{
padding 10px 5px;
}


margin Property

It sets the margin of a certain element.

h1{
margin: 100px;
}


font-family Property

Used to tell the browser to use the given fonts. If the browser doesn't support the current font, it will try the other.

html{
font-family: helvetica, arial, sans-serif;
}


overflow Property

Used to clip or add scroll bars when the element gets too big to fit in a given area.

div{
width:100px;
height:50px;
overflow: auto;
}


background-color Property

Changes the background color of an element. Can also input RGB values or HEX codes instead of the color name.

body{
background-color: blue;
}


text-align Property

Used to properly align the element in a horizontal manner.

h1{
text-align: center;
}


font-size Property

Changes the size of the font.

p{
font-size: 20px;
}
